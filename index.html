ECW Document Labeling Master Template

Standard format: MM/DD/YYYY – [CATEGORY] – Description – Facility

Jump to: RESULT ORDER MED PA-MED PA-IMG CONSULT RECORDS DME REFERRAL HOSP/ER

RESULT

Used for lab and imaging results and other clinical results requiring review or filing.

Document Type Category Tag Label Format File To Notes / Action
Lab Result (non-electronic) RESULT MM/DD/YYYY – RESULT – Lab – Facility Lab Documents Attach to existing order if present; route to provider/CS as needed.
Imaging Result (no originating order) RESULT MM/DD/YYYY – RESULT – Imaging – Facility Imaging (XR/US/CT/MRI) File only unless ordered by our provider.
Positive Cologuard RESULT MM/DD/YYYY – RESULT – Positive Cologuard Lab Documents Tag HP; notify via Teams.

ORDER

Used for any document reflecting an order or requiring provider signature.

Document Type Category Tag Label Format File To Notes / Action
PT/OT/HH Order ORDER MM/DD/YYYY – ORDER – HH Order # – Facility PT/OT/HH Orders Tag HP if signature needed.
Plan of Care (POC) ORDER MM/DD/YYYY – ORDER – POC – Facility PT/OT/HH Orders Route for signature as required.
Pre-Op Request ORDER MM/DD/YYYY – ORDER – Pre-Op Request – Facility Pre-Op Tag HP.
