📠 How to Use the Fax Routing Tool

Get routing instructions for any fax in 2 clicks or less. No guessing, no asking, no delays.

How to Use It

1
Identify the document type

Look at the fax — is it a hospital record? A lab result? A PA request? Pick the matching category on the left panel.

2
Click the category — then the document type

If only one item is in a category, it opens instantly. If multiple, a list drops down — pick the match. The colored pill next to each shows the action at a glance.

DEL = delete · FILE = file & route · TE = create TE · SNIP = snip & send to chat · ESC = escalate
3
Follow the action banner

The big colored banner tells you exactly what to do. Document Name, Document Folder, and Assigned To are shown below it.

4
Fax to Electronic conversion request?

Found under Signature Required. Snip the document and send to the team chat tagging @Megan and @Jaimie. No filing required — Level 1 action.

5
Not sure? Use Triage or Search

Click "Not sure what this is?" for a guided yes/no walkthrough. Or type a keyword in the search bar — "discharge", "lab result", "PA denial".

Press / on your keyboard to jump straight to search. Press Esc to close anything.

📌 General Rules & Reminders

Applies to every fax, every time. These do not live on individual cards — they apply across the board.

TEMPORARY BACKLOG PROTOCOL — IN EFFECT UNTIL FURTHER NOTICE

Batch processing by category is not possible right now. Doc Category is not auto-populating and sender fax sorting is broken. The only usable sort is Date.

Sort by Date ascending. Work oldest first. One touch per fax — make the call and move on.

How to set up your view
1. Sort by DATE column — ascending (oldest first)
2. Work top to bottom — do not skip, do not come back
3. One decision per fax — see the decision tree below
When you open each fax — ask in this order:
STEP 1
Is it STAT, a discharge, ER record, or imaging result?
Yes — process immediately. Mark HP. Do not set aside. Same-day, no exceptions.
No — move to Step 2.
STEP 2
Is it a delete?
Active provider Rx, Tango, non-adherence, confirmation of receipt, key listed PA, duplicate — delete it now and move on.
No — move to Step 3.
STEP 3
Which bucket does it belong to?
Pick the closest category using the left nav. Attach, label, route. One touch — done. Move to the next fax.
STEP 4
Still not sure?
File to *Important Notifications, create TE, assign to Clinical Admin. Do not hold it. Keep moving.
Golden Rule for the Backlog
Every fax gets one decision. You do not go back. You do not hold. You do not skip. Pick the closest category, route it, and move to the next one. Speed and consistency beats perfection right now.

🚨 Imaging Result Processing — Mandatory Rules

⚡ STAT / CXR / CT Chest / Any Acute Study = SAME DAY
Cannot process same-day → escalate to lead immediately. No exceptions.
Check 1 — Verify Ordering Provider
Pull up the patient in eCW. Confirm the ordering provider on the faxed result matches what is in the system. If their name is in eCW they are an internal provider — do NOT mark FYI.
Check 2 — Label Correctly
Imaging results are not consult notes. Label as the correct document type: X-ray, CT, MRI, Ultrasound. File to *Imaging – Xray/US/CT/MRI.
Check 3 — Route to Ordering Provider
After correct labeling, route to the provider — not records only.
Processing SLA
STAT / urgent imaging → Same day
Routine imaging → 24–48 business hours from receipt
Beyond SLA → Lead is notified and manually processes it

🎯 One-Touch Goal

Every document should be correctly categorized, routed once, and not require rework.
Route it right the first time. That is the only metric that matters at Level 1.

🔧 Fix Before Routing

🔧
If something is missing or needs a small correction — fix it before routing. Do NOT send to another team just to fix small issues. One touch, one action.

⚑ HP Rules

⚑
Hospital / ER is ALWAYS HP. No exceptions. Do not delay for review.
⚑
Mark HP if it is noted on the form — FMLA, PA requests, imaging orders, DME, lab requests.
⚑
STAT = HP. If the fax says STAT anywhere — it is HP.
⚑
CGM, end of life, and nebulizers under DME — always HP.

📅 Date Rules

📅
For imaging PAs — use the order date, not the fax received date.
📅
For signature documents — use the date on the document, not today's date.
📅
When in doubt — use the date that appears on the actual document.

🚫 Hard Stops

🚫
ER/Hospital follow-up — 3 attempt rule. If unable to reach patient after 3 attempts, send TE to Office Manager as FYI. Do not keep attempting without escalating.
🚫
Results for an out-of-office provider — do not leave in queue. Route to the Office Manager to reassign or hold for provider return.
🚫
DME with a signature line — signature rule wins. Route directly to provider, no Clinical Admin needed.
🚫
No multiple actions for the same issue. Denial, appeal, follow-up — all go in the same existing action. Do not create a new one.
🚫
Do not send to a provider unless it is clearly a provider decision. Providers are not triage.
🚫
Do not guess between categories. When unsure → Still Confused → Clinical Admin.
🚫
Failure to Schedule with Specialist is NOT a delete. Route it to Referrals for follow-up.
